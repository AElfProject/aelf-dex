# Front

## Development

1. set your AElf chain node URL in the file `src/common/config.js`;
2. install dependencies
    ```shell script
    yarn
    ```
3. run local dev server
    ```shell script
    npm run dev
    ```
4. Coding in `src/`

## Production build

run
```shell script
npm run build
```
in `aelf-dex/web/front`, and get the output in the directory `aelf-dex/web/front/dist`
